# ai-pm-learning-agent
# 🤖 AI PM 学习助手

> 一个飞书机器人，帮你把碎片时间看到的好文章自动分析、分类、存入飞书多维表格知识库。

专为**碎片化学习**设计：发一个链接，AI 自动帮你判断这篇内容值不值得深读、应该怎么用、存到哪里。

---

## 🎯 这个项目解决什么问题

你是否有这样的困境：

- 看到好文章随手收藏，但收藏夹从来不打开
- 不知道这篇内容对自己有没有用，懒得细读
- 知识散落在各处，无法形成体系

这个助手的逻辑是：**看到好内容 → 发链接给飞书机器人 → AI 自动分析价值、分类入库 → 知识沉淀在飞书表格里随时查阅**。

---

## 🏗️ 系统是怎么工作的

用一张图来解释：

```
你发消息给飞书机器人
        ↓
飞书把消息转发给你的服务器（Webhook）
        ↓
服务器用 Jina.ai 抓取链接内容
        ↓
把内容发给 DeepSeek AI 分析
        ↓
AI 返回：分类建议 + 摘要 + 实战任务
        ↓
服务器把分析结果回复给你的飞书
        ↓
你回复"1"确认入库
        ↓
服务器把内容写入飞书多维表格
```

**技术上这是一个"后端服务"**，没有任何界面，你看到的所有交互都在飞书里完成。

**各个部分的角色：**

| 组件 | 作用 |
|------|------|
| 飞书机器人 | 你的交互界面，发消息、收回复都在这里 |
| Flask（Python） | 后端服务器，接收飞书的消息、处理逻辑 |
| Jina.ai | 帮你抓取网页内容，绕过直接请求的限制 |
| DeepSeek AI | 分析内容价值，生成摘要和分类建议 |
| 飞书多维表格 | 你的知识库，所有内容最终存在这里 |
| Replit | 运行服务器的地方（也可以换成其他云服务） |
| UptimeRobot | 每5分钟 ping 一次服务器，防止它休眠 |

## ✨ 功能介绍

### 三个知识库

| 库名 | 存什么 |
|------|--------|
| ⚡ 快读资讯库 | 行业动态、资讯，了解一下就够 |
| 🔥 上手实战库 | 有具体方法论，现在就能动手做 |
| 📦 待激活库 | 有价值但暂时用不上，等条件具备再用 |

### 使用方式

| 你发什么 | 机器人做什么 |
|----------|------------|
| 一个网页链接 | 自动抓取分析，给出分类建议和摘要 |
| `1` | 按 AI 建议入库 |
| `快读` | 强制存入快读资讯库 |
| `上手` | 强制存入上手实战库 |
| `存档` | 强制存入待激活库 |
| `整合` | 生成知识整合报告，看你最近学了什么 |
| 粘贴文字 | 直接分析文字内容（适合小红书、公众号） |

> **小红书 / 公众号怎么用：** 这两个平台有反爬机制，无法自动抓取。你可以用其他 AI 工具把图片转成文字，然后粘贴给机器人，机器人会继续分析。

### 知识图谱

每次入库后，系统会自动更新知识图谱，记录你在每个方向上积累了多少内容、掌握程度如何。发送`整合`可以看到完整的学习报告。

## 🚀 部署教程

### 第一步：准备飞书机器人

1. 去[飞书开放平台](https://open.feishu.cn/)创建一个自建应用
2. 开启**机器人**能力
3. 开启**接收消息**事件订阅
4. 记录下 `App ID` 和 `App Secret`

### 第二步：准备飞书多维表格

在飞书里创建一个多维表格，建以下4个子表：

**快读资讯库**，需要这些列：
- 标题（文本）
- 高价值总结（文本）
- 能力大类（文本）
- 子方向（文本）
- 原文链接（超链接）

**上手实战库**，需要这些列：
- 标题（文本）
- 高价值总结（文本）
- 能力大类（文本）
- 子方向（文本）
- 实战任务（文本）
- 内化状态（文本）
- 原文链接（超链接）

**待激活库**，需要这些列：
- 标题（文本）
- 内容摘要（文本）
- 能力大类（文本）
- 子方向（文本）
- 激活条件（文本）
- 原文链接（超链接）

**知识图谱**，需要这些列：
- 能力大类（文本）
- 子方向（文本）
- 掌握程度（文本）
- 内容数字（数字）
- 关联项目（文本）
- 下一步（文本）
- 最近更新（文本）

记录下多维表格的 `App Token` 和每个子表的 `Table ID`（在表格 URL 里可以找到）。

### 第三步：获取 DeepSeek API Key

去 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册，获取 API Key。

### 第四步：部署代码

**Fork 这个仓库**，然后在 Replit 或其他云服务上运行。

需要配置以下环境变量：

```
FEISHU_APP_ID=你的飞书App ID
FEISHU_APP_SECRET=你的飞书App Secret
FEISHU_APP_TOKEN=多维表格的App Token
TABLE_ID_FAST=快读资讯库的Table ID
TABLE_ID_DEEP=上手实战库的Table ID
TABLE_ID_HOLD=待激活库的Table ID
TABLE_ID_GRAPH=知识图谱的Table ID
DEEPSEEK_API_KEY=你的DeepSeek API Key
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

### 第五步：配置飞书 Webhook

在飞书开放平台的事件订阅里，填入你的服务器地址：

```
https://你的服务器地址/webhook
```

### 第六步：保持服务器在线（可选）

如果用 Replit 免费版，服务器会休眠。用 [UptimeRobot](https://uptimerobot.com/) 每5分钟 ping 一次你的地址，可以保持服务器持续在线。

## 🔧 个性化配置

打开 `main.py`，找到 `USER_PROFILE` 这一段，把里面的内容改成你自己的背景：

```python
USER_PROFILE = """
用户背景：
- 身份：你是谁，做什么的
- 目标行业：你关注哪些行业
- 学习阶段：你现在在学什么
- 实战方向：你想做什么项目
"""
```

改完之后，AI 会根据你的背景来判断内容对你的价值，分析结果会更准确。
